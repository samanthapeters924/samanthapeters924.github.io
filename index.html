  Projects

Workable- Web application design

At Startup Institute, I worked on a 4 person team consulting on Workable's new offer letter generator tool.

I collaborated with my own team, as well as with Workable's internal team. We also spoke with potential users. We ultimately decided on building an offer letter tool that would link in from content rich landing pages.

Workable- Web Application Design

After creating initial wireframes, I created a prototype for the offer letter generating tool. This is a mock-up of a landing page, that would then link to the actual tool.

We wanted to capture multiple search terms, and provide relavent information. Users could then access the offer letter generating tool.

Workable- User Flow

We wanted the offer letter tool to drive users to a free trial. Workable's new background check tool was a good 'next step'.

This flow demostrates how users would go from using the offer letter gererator, to signing up for a free trial in order to use the background check feature

Blur- Marketing e-mail design
